# 💳 Credit Analysis System (Python)

This project demonstrates the evolution of a credit analysis system built in Python, focusing on **code quality, readability, and scalability**.

It was developed as an **academic project**, showcasing best practices such as refactoring, modularization, and object-oriented programming.

---

## 📚 Purpose

The main goal is to present different approaches to solving the same problem:
**credit approval analysis based on predefined business rules**.

Each version improves the previous one, demonstrating how code can evolve from a simple structure to a more maintainable and professional design.

---

## 🧱 Project Structure

```bash
/credit_analyze
│
├── /src
│   ├── __init__.py           # Marks src as a Python module
│   ├── nested_logic.py       # Version 1 - deeply nested conditionals
│   ├── function_logic.py     # Version 2 - functional refactor (early returns)
│   ├── class_logic.py        # Version 3 - object-oriented implementation
│   └── dataclass_logic.py    # Version 4 - optimized with @dataclass
│
├── /tests
│   ├── __init__.py           # Marks tests as a Python module
│   └── test_analysis.py      # Test scenarios for credit validation
│
└── README.md                 # Project documentation
```

---

## 🔄 System Evolution

### 🔹 Version 1 — Nested Logic

* Uses deeply nested `if` statements
* Difficult to read and maintain

### 🔹 Version 2 — Functional Approach

* Introduces helper functions
* Applies **early return (fail-fast)**
* Improves readability

### 🔹 Version 3 — Object-Oriented Programming

* Introduces the `Client` class
* Encapsulates data and behavior
* Improves organization and scalability

### 🔹 Version 4 — Dataclass Optimization

* Uses `@dataclass` to reduce boilerplate
* Cleaner and more declarative code
* Maintains the same business logic

---

## ⚙️ Business Rules

The system evaluates credit approval based on:

* Minimum age: **18 years**
* Minimum income: **2000**
* Credit score tiers:

  * **≥ 700** → Approved with great conditions
  * **≥ 500** → Conditional approval
  * **< 500** → Rejected
* Clean credit history
* Availability of collateral
